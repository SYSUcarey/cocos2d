## 说明

2. 项目环境：
* cocos2d-x-3.16

* python 2.7.15

***

## 要求

实现一个横版游戏，具体要求：

* 左边wasd4个虚拟按键能控制角色移动

* 右边2个虚拟按键x，y能控制角色播放不同的帧动画

* 界面所有字体要求：使用fonts目录下的arial.ttf，字体大小为36

* 角色不会移动到可视窗口外

* 添加倒计时

* 添加人物血条

* X、Y播放的动画不能同时播放

* 点击虚拟按键x播放帧动画并让血条减少,点击y播放帧动画并让血条增加（加分项）
